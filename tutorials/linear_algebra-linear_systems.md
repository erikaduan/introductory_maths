# Linear systems

There is a fundamental relationship between equations and lines. Lines
represent the vector subspace of all solutions found for a linear
equation.

Depending on the dimension of the vector space, we can describe linear
equations with some geometric intuition. For example,
![3x_1 = 9](https://latex.codecogs.com/svg.format?3x_1%20%3D%209 "3x_1 = 9")
is solved as
![x_1 = 3](https://latex.codecogs.com/svg.format?x_1%20%3D%203 "x_1 = 3"),
where ![x_1](https://latex.codecogs.com/svg.format?x_1 "x_1") describes
a point (a copy of
![\\mathbb{R}^0](https://latex.codecogs.com/svg.format?%5Cmathbb%7BR%7D%5E0 "\mathbb{R}^0")
inside
![\\mathbb{R}^1](https://latex.codecogs.com/svg.format?%5Cmathbb%7BR%7D%5E1 "\mathbb{R}^1")).
